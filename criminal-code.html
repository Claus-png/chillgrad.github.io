Уголовный кодекс

Система правовой ответственности за дестабилизирующую активность на территории Королевства.

ТЕРМИНЫ

«Дестабилизирующая активность» (ДА) — любые действия, создающие избыточный шум, панику, принудительную срочность или иную нагрузку, нарушающую нормативно-правовой режим стабильности («чилл»).
«Суета» — народный синоним ДА, официально закреплён в судебной терминологии (Статья 9 Конституционного Акта).

// Коэффициенты

Поправочные коэффициенты формулы СДА

СДА = Тариф × (1 + Kшум + Kмасштаб + Kрецидив)
Kшум

(дБ_факт – дБ_норма) / 10
Макс. значение: 5,0

Kмасштаб

log₁₀(число затронутых граждан)
Макс. значение: 3,0

Kрецидив

0 / 0,5 / 1,0 / 2,0
(1 / 2 / 3 / 4+ нарушения/год)

// Судопроизводство

Система Arbiter-Assist 1.0

Для кластеризации дел и поиска прецедентов используется система Arbiter-Assist 1.0. ИИ-система анализирует базу судебных решений, предлагает аналогичные прецеденты и вычисляет базовый размер СДА. Окончательный вердикт выносится исключительно судьёй-человеком — использование ИИ для принятия итогового решения категорически запрещено.
